# NLP Prose Utils

**Python tooling for prose analysis:** readability metrics, lexical diversity, TF-IDF
extraction, summarization, cliche detection, and text-matching search.

## Features

- **Readability** — Kincaid, ARI, Coleman-Liau, Flesch, Fog, SMOG
- **Lexical diversity** — Yule I, word entropy, synonym suggestions
- **Description** — hashtag suggestions, TF-IDF keywords, extractive summary
- **Search** — Rabin-Karp and wildcard pattern matching, cliche detection

## Quick start

Requires Python 3 with NLTK, NumPy, and scikit-learn.

```bash
python main.py < sample.txt
```
